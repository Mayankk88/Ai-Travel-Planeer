# Commands to Deploy to kkhushi-07's GitHub

Run these commands one by one in PowerShell/Terminal:

# Step 1: Add remote (if not already added)
git remote add khushi https://github.com/kkhushi-07/AI-Travel-Planner.git

# Step 2: Push to kkhushi's repo
git push khushi main --force

# When it asks for credentials:
# Username: kkhushi-07
# Password: [kkhushi-07's GitHub Personal Access Token]

# To create Personal Access Token:
# 1. kkhushi-07 logs into GitHub
# 2. Go to: https://github.com/settings/tokens
# 3. Click "Generate new token (classic)"
# 4. Select "repo" scope
# 5. Generate and copy token
# 6. Use token as password when pushing

# Alternative: Use GitHub CLI
gh auth login
git push khushi main --force
